Curva de valor

Onde o SalasUFS Gestor se posiciona

Cada fator abaixo representa uma dimensão pela qual soluções de automação de ambientes competem hoje. A curva do SalasUFS Gestor (em destaque) eleva fatores negligenciados pelos concorrentes — integração multi-ambiente, indicadores de BI e escalabilidade institucional — enquanto elimina a dependência de intervenção manual.

Matriz ERRC

Eliminar · Reduzir · Elevar · Criar

A estrutura das quatro ações traduz a curva de valor em decisões concretas de projeto para o módulo Gestor do SalasUFS.

Eliminar

O que deixamos para trás

  • Notificação manual dependente da ação do usuário
  • Dependência de crachá/RFID físico para acionamento
  • Soluções isoladas por ambiente único
Reduzir

O que simplificamos

  • Complexidade de instalação de hardware dedicado
  • Custo de implantação por ambiente automatizado
  • Ajustes manuais repetitivos de temperatura
Elevar

O que ampliamos

  • Nível de integração entre ambientes, blocos e campi
  • Qualidade dos indicadores de consumo energético
  • Acesso via navegadores Web e dispositivos móveis
Criar

O que é inédito

  • Painel unificado de indicadores em Power BI
  • Importação automatizada de horários de funcionamento
  • Disponibilização do software para outras instituições
