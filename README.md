# SalasUFS Gestor — Blue Ocean Strategy Canvas

> Visualização estratégica do projeto **SalasUFS Gestor**, desenvolvido no âmbito do PIBITI 2024/2025 (UFS), posicionando a solução frente aos trabalhos correlatos em automação de ambientes universitários.

🔗 **[Acessar o Strategy Canvas ao vivo](https://SEU-USUARIO.github.io/NOME-DO-REPO/)**

---

## 📌 Sobre o projeto

O **SalasUFS Gestor** é um software para gestão integrada de ambientes universitários automatizados via IoT — controlando luzes e ares-condicionados, agrupando ambientes por blocos e campi, e disponibilizando indicadores de consumo energético em um painel Power BI, acessível via Web e Mobile.

Este repositório contém uma página interativa que aplica a ferramenta **Blue Ocean Strategy Canvas** para posicionar o projeto frente a três trabalhos correlatos identificados na literatura:

| Trabalho | Foco |
|---|---|
| Cunha (2018) | Monitoramento de temperatura com notificação manual |
| Huang et al. (2019) | Automação completa de uma sala de aula inteligente |
| Ugale et al. (2019) | Automação de laboratórios via RFID |

## ✨ O que a página mostra

- **Curva de valor animada** — comparação em 7 fatores de competição (automação, integração multi-ambiente, indicadores de BI, acesso web/mobile, escalabilidade, adoção e custo-benefício), com as linhas se desenhando ao carregar.
- **Matriz ERRC** (Eliminar · Reduzir · Elevar · Criar) — traduzindo a estratégia em decisões concretas de projeto, revelada com animação ao rolar a página.
- **Referências** aos trabalhos correlatos utilizados na análise.

## 🛠️ Stack

- HTML, CSS e JavaScript puros (sem dependências externas além de fontes do Google Fonts)
- SVG para o gráfico de curva de valor, com animação nativa via `stroke-dashoffset`
- `IntersectionObserver` para as animações de scroll
